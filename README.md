# Phishing Website Detection

## Overview
This project is a phishing website detection web application developed using Python and Flask. It helps identify whether a website is legitimate or a phishing website.

## Features
- Detects phishing websites
- Simple and user-friendly interface
- Built using Python and Flask

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run
1. Install the required Python packages.
2. Run:
   python app.py
3. Open your browser and visit:
   http://127.0.0.1:5000/

   ## Installation

1. Install the required package:
   ```
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   python app.py
   ```

3. Open your browser and visit:
   ```
   http://127.0.0.1:5000/
   ```
